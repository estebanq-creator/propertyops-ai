## Technical Notes

### API Endpoints Used

**Stripe:**
- `GET /v1/account` - Account info
- `GET /v1/balance` - Current balance
- `GET /v1/payouts` - Payout history
- `GET /v1/balance/history` - Balance transactions

**Mercury:**
- `GET /api/v1/organizations/{id}` - Org info
- `GET /api/v1/organizations/{id}/accounts` - Account list
- `GET /api/v1/organizations/{id}/accounts/{id}/balance` - Balance
- `GET /api/v1/organizations/{id}/accounts/{id}/transactions` - Transactions

### Rate Limits

- Stripe: ~100 requests/sec (test mode may be lower)
- Mercury: ~10-100 requests/min (check docs)
- Scripts use batch fetching to minimize calls

### Error Handling

- All scripts exit with code 1 on failure
- Error messages include actionable guidance
- Error reports saved to output directory
