## Paperclip Adapter Configuration Issue (Resolved)

**Problem:** Agent config threw error "Instructions bundles are only available for local adapters"

**Root Cause:** Using OpenClaw gateway (remote/HTTP adapter at `http://127.0.0.1:3100`) with `instructionsFilePath` field in adapter config. Remote adapters cannot read local filesystem files.

**Resolution:** 
- Remove `instructionsFilePath` from agent adapter configuration
- Replace with inline `promptTemplate` field, or pass instructions at session creation time
- Local adapters (e.g., `claude_local`) support `instructionsFilePath`; remote adapters do not

**Esteban's Setup:**
- Paperclip profile: `openclaw-local`
- API base: `http://127.0.0.1:3100`
- Adapter type: HTTP/remote (not local process)

**Reference:** Agent configs likely in Paperclip UI → Agent Configuration section
